# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this project, please do not open a public issue.

Please report it privately by contacting the maintainer directly.

When reporting a vulnerability, include:
- A clear description of the issue
- Steps to reproduce it
- The affected file or function, if known
- Any potential impact you observed

I will review reports as quickly as possible and work to validate and address confirmed issues.

## Scope

This repository is intended for defensive security validation and safe testing of Microsoft Defender for Endpoint controls.

Please do not use this project to:
- perform unauthorized testing
- deploy harmful payloads
- evade security controls
- simulate malicious activity outside of approved lab or enterprise environments

## Supported Versions

At this time, only the latest public version of this repository is considered supported.
